Esse é um projeto didático que busca criar uma ferramenta de aprendizado visual e prático para o modelo aditivo de cores RGB . Para isso, desenvolveu-se um software de calculadora de cores que utiliza este modelo para seu funcionamento. Nele as cores são criadas pela soma das intensidades das três cores primárias, vermelho (Red), verde (Green) e azul (Blue), cada cor é definida por valores numéricos de 0 a 255 para cada componente (R, G, B) em um plano cartesiano de 3 dimensões (fig 1). Isto é possível pela forma que o cérebro enxerga as cores e pelo princípio de independência dos raios de luz .

Para o desenvolvimento do site, inicialmente foi elaborado um wireframe, esboço básico que define a estrutura de cada página, com a disposição de elementos como cabeçalhos, seções, botões e menus. Com o layout definido, passou-se para a criação do design visual, estabelecendo a paleta de cores, tipografia, ícones e o estilo dos elementos visuais. Após essa etapa, utilizou-se o VSCode (Fig 1) para a criação das tecnologias de front-end que estruturam o site, HTML para estruturar o conteúdo, CSS para estilizar os elementos e JavaScript para adicionar interatividade. Neste caso específico, não foram utilizadas tecnologias de back-end.

O protótipo de máscara de palhaço foi criado a partir de uma máscara branca de papel. Ela foi pintada com cores referentes ao sistema RGB, e um nariz de palhaço foi anexado para realçar suas características. Em seguida, foi confeccionado um disco de Newton, que foi acoplado a um rolamento na parte traseira da máscara, permitindo que o disco gire e crie o efeito visual de mistura de cores.

O site teve êxito em disponibilizar uma saída de cor para uma entrada de coordenadas cartesianas RGB e uma saída de coordenadas cartesianas RGB para uma entrada de cor como proposto inicialmente pelo projeto e a máscara (Fig 2) também teve êxito na sua atuação como um disco de Newton

Esse é um projeto didático que busca criar uma ferramenta de aprendizado visual e prático para o modelo aditivo de cores RGB . Para isso, desenvolveu-se um software de calculadora de cores que utiliza este modelo para seu funcionamento. Nele as cores são criadas pela soma das intensidades das três cores primárias, vermelho (Red), verde (Green) e azul (Blue), cada cor é definida por valores numéricos de 0 a 255 para cada componente (R, G, B) em um plano cartesiano de 3 dimensões (fig 1). Isto é possível pela forma que o cérebro enxerga as cores e pelo princípio de independência dos raios de luz .

A luz viaja do olho para a retina, localizada na parte de trás do olho. A retina é coberta com milhões de células receptivas de luz chamadas bastonetes e cones (fig 2). Os humanos, como tricromatas, possuem 3 tipos de cones, o de comprimento de onda curto (S), médio (M) e longo (L), quando essas células detectam luz, elas enviam sinais para o cérebro.

A estimulação diferencial deles cria eixos de cores entre vermelho,verde e azul, permitindo a visualização de cerca de 10 milhões de cores, todas derivadas de dois tipos de mistura de luz a aditiva e subtrativa (fig 3), a mistura aditiva envolve a adição de componentes espectrais já subtrativa a absorção de partes do espectro.

O círculo de cores (fig 4), criado pela primeira vez por Newton é uma representação visual simples e direta dos princípios que também fundamentam o sistema aditivo de cores RGB. Ambos ilustram como a combinação de diferentes luzes pode criar novas cores, com aplicações que vão desde demonstrações didáticas até tecnologias avançadas de exibição de imagem.

Além disso, o princípio da independência dos raios de luz permite que cada cor seja representada como um ponto em um plano cartesiano tridimensional, pois cada cor pode ser considerada separadamente e combinada sem interferência, facilitando a visualização e manipulação das cores em modelos aditivos como o RGB.
